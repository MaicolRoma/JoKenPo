# JoKenPo
Juego de Piedra, Papel y Tijera 

<img width="1915" height="988" alt="Captura desde 2026-06-29 19-58-12" src="https://github.com/user-attachments/assets/ccd1211b-6347-4de0-aff0-f8316a3d58f0" />

Este es el antiguo y bien conocido juego de piedra, papel y tijera, 
con algunas animaciones funcionalidades para hacerlo mas entretenido.


## link: https://maicolroma.github.io/JoKenPo/

## Funcionalidades:
- Contador de las Rondas
- Contador de las Rachas
- Contador de Puntuacion
- Reiniciar Partida
- Activar Sonidos o Desactivar
- Panel de historial de rondas

## Herramientas y Tecnologias Usadas:
- HTML
- CSS
- Javascript
